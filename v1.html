Goal is to create a simple, reusable, json passable radar chart that takes in the following values as an input, and have the ability to render multiple radar charts, as well as have the ability to drag and see the data manipulation in real time as the values are updated.

Code: 
//Data Example
var data = [{
  "className": "Name of the Class",
  "normalize": true,
  "scale_max": 20,
  "drag": true
  "axes" : [
    {
      "axis" : "name of axis",
      "value": 4,
      "abs_min" : 0,
      "min" : 0,
      "abs_max" : 10,
      "max" : 10,
      "step": 1
    },...
  ]
}]

RadarChart.draw("#chart", data);
